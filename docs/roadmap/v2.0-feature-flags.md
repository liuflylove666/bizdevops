## 5. Flag 生命周期

1. **新增**：随 Epic PR 引入，默认 `false`
2. **灰度**：按本文件第 3 节节奏开启，先白名单 → 10% → 50% → 100%
3. **转正**：Sprint 7 / v2.0 GA 后转为默认 `true`
4. **清理**：GA 后 30 天，删除代码分支与 Flag 定义，结束生命周期
